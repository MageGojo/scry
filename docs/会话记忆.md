## 用户目标
- 做一个 Burp/Reqable 式的抓包 + 渗透套件(Rust workspace + gpui 界面,复用 `mage_ui`),优先 macOS。
- **不抢系统代理**:配合 Proxifier 按进程喂流量;也能像 Wireshark 那样直接抓本机网卡。
